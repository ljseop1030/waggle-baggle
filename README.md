# 🗣️ 와글바글

**실시간 캠퍼스 학습 공간 혼잡도 안내 서비스**

> “도서관 가기 전에, 자리가 있는지 미리 알 수는 없을까?”


## 📌 프로젝트 소개

**와글바글**은 교내 도서관·학습 공간의 혼잡도를  
**실시간으로 확인할 수 있는 캠퍼스 연동 서비스**입니다.

Beacon(BLE) 기반 자동 감지 시스템을 활용해  
학생들의 불필요한 이동을 줄이고, 학습 효율을 높이는 것을 목표로 합니다.


## 🚀 확장 계획

- 📍 사용자 위치 기반 **좌석 추천**
- 🎉 교내 행사 및 공지 알림 연계
- 🏫 도서관 → 학생회관 → 전공 건물 등 단계적 확대


## 📎 참고

- 본 프로젝트는 **창업동아리 중간평가 발표**를 기반으로 제작되었습니다.
- Beacon은 Bluetooth Low Energy(BLE) 기술을 사용합니다.


✨ **와글바글로 더 스마트한 캠퍼스 라이프를 만들어갑니다.**
